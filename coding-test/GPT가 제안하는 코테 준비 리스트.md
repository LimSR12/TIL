# 🟢 Tier 1 — 무조건 나옴 (코테 기본 체력)

## 1. 자료구조 기반 문제 해결

- 배열 / 문자열
- 스택
- 큐
- 덱 (Deque)

👉 활용 유형

- 괄호 검사
- 단조 스택 (ex. 오큰수, 탑, 3015 같은 문제)
- 슬라이딩 윈도우
- 시뮬레이션

---

## 2. DFS / BFS

그래프 문제가 아니어도 상태 탐색 문제 = DFS/BFS 입니다.

👉 필수 응용

- 완전 탐색
- 백트래킹
- 연결 요소 탐색
- 최단 거리 (BFS)

---

## 3. 정렬 + 구현

- Custom 정렬
- Comparator 활용
- 좌표 정렬
- 이벤트 정렬

👉 대부분 기업 코테 1~2번 문제는 여기서 나옵니다.

---

## 4. 그리디

“지금 최선이 전체 최선인가?” 판단 능력

👉 대표 유형

- 회의실 배정
- 동전 문제
- 구간 선택
- 정렬 기반 greedy

---

## 5. 투 포인터 / 슬라이딩 윈도우

👉 시간복잡도 개선 핵심 패턴

- 부분합
- 연속 구간
- 조건 만족 구간 찾기

---

# 🟡 Tier 2 — 중급 필수 (합격권 진입)

## 6. 이분 탐색 (Binary Search)

단순 탐색이 아니라

👉 Parametric Search

- 최소값 최대값 찾기
- 조건 만족 여부 판별

코테 출제율 매우 높음

---

## 7. 그래프 기본

- 인접 리스트
- 방문 처리
- 사이클 탐지

---

## 8. DP (Dynamic Programming)

많이 나오진 않지만  
한 번 나오면 당락 갈립니다

👉 필수 유형

- 1차원 DP (계단, LIS)
- 2차원 DP (경로 문제)
- 점화식 설계 능력

---

## 9. 구현 시뮬레이션

삼성 / 카카오 스타일

- 방향 이동
- 회전
- 상태 변화

(이건 알고리즘보다 코딩 능력 테스트)
