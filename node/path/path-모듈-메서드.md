| 메서드              | 역할                                              | 핵심 포인트                                                     |
| ------------------- | ------------------------------------------------- | --------------------------------------------------------------- |
| `path.resolve()`    | 상대/절대 경로를 입력했을 때 **절대 경로로 변환** | `..`, `.` 등을 해석하여 실제 파일 시스템에서의 절대 위치로 확정 |
| `path.isAbsolute()` | 문자열이 **절대 경로인지 판별**                   | 입력이 `/root/...` 또는 `C:\...` 형태인지 확인                  |
| `path.normalize()`  | 경로에서 `..`, `.`, 중복 `/` 등을 정리            | `C:/test/../dir` → `C:/dir` 같은 표준 형태로 변환               |
